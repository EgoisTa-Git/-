# Раскрутим планету

Скрипт-пример работы с терминалом и управляющими символами ANSI.

## Установка

- Для корректной работы Вам необходим Python версии 3 и выше.

## Запуск

Для запуска скрипта введите в консоли:

```shell
python main.py
```
